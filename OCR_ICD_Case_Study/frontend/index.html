Pharma ICD OCR Demo (LangChain + Azure OCR + OpenAI + pgvector)

Workflow:
1. Upload an image (prescription / discharge summary).
2. Backend runs OCR (mock or Azure).
3. LLM extracts ICD codes + supporting sentences.
4. LangChain + FAISS/pgvector ground those sentences to page + bounding box.
5. Grey boxes = all OCR lines, Red boxes = ICD supporting lines.
